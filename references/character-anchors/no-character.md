# No-Character Mode (无角色模式 / 纯概念模式) — 无 IP 形象生成规范

## 概述

无角色模式不为配图绑定任何具象化角色或 IP 形象。画面完全由抽象概念元素、场景氛围、物体关系或视觉结构本身来传达文章核心思想。适合追求纯粹概念表达、数据可视化、方法论说明、氛围营造的场景。

## 核心原则

- **概念即画面**：画面本身就是概念的视觉化，不需要角色作为中介
- **元素即语言**：用形状、线条、空间、光影、物体关系来"说话"
- **氛围即情绪**：通过色调、构图、留白传递文章的情绪基调
- **结构即逻辑**：用视觉结构（层级、流向、对比、映射）表达文章逻辑
- **风格即契约**：同一文章的所有插图必须使用**完全相同的视觉风格**（Step 5 选定后锁定，不可中途更换）

## 三种视觉方向

### 方向 A：纯概念隐喻（Abstract Metaphor）

画面只有抽象元素，用视觉结构本身讲故事。

| 文章概念 | 视觉表达 | 元素构成 |
|---------|---------|---------|
| 数据瓶颈 | 一个漏斗，上方大量光点，下方只有少量通过 | 几何形状 + 粒子 |
| 增长曲线 | 一条由藤蔓构成的上升线，节点是果实 | 有机线条 + 节点 |
| 系统崩溃 | 一座由积木搭成的塔，中间几块正在滑落 | 几何体 + 动态张力 |
| 信息过载 | 一个容器正在溢出，内容物是文字碎片 | 容器 + 溢出效果 |

**特点**：高度抽象，一图一概念，没有具象物体。

### 方向 B：物体叙事（Object Narrative）

画面以日常物品或自然元素为主体，通过物品之间的关系、状态、变化来表达概念。

| 文章概念 | 视觉表达 | 物体构成 |
|---------|---------|---------|
| 供需缺口 | 一排空椅子 vs 拥挤的人群 | 家具 + 人群剪影 |
| 技术迭代 | 一把旧钥匙正在打开一扇新门 | 旧物 + 新空间 |
| 认知断层 | 一本书被撕成两半，中间是深渊 | 书籍 + 断裂 |
| 能量转化 | 一个灯泡正在吸收周围的黑暗 | 光源 + 环境 |

**特点**：有具象物体，但物体是"概念载体"，不是角色，无表情、无动作拟人化。

### 方向 C：场景氛围（Scene Atmosphere）

纯环境/场景，通过空间关系、光影变化、氛围营造来传递情绪和信息。

| 文章概念 | 视觉表达 | 场景构成 |
|---------|---------|---------|
| 市场寒冬 | 一片被雪覆盖的废墟，远处有微弱的灯光 | 环境 + 光线对比 |
| 创新萌芽 | 一片荒芜的土地上，一株幼苗正在破土 | 自然 + 生命力 |
| 信息迷雾 | 一条被浓雾笼罩的道路，路标模糊不清 | 道路 + 气象 |
| 技术深渊 | 一个深不见底的竖井，井壁是代码流 | 空间 + 数字纹理 |

**特点**：无主角，画面本身是"舞台"，概念通过环境氛围传递。

## 生成方法

### 方法一：概念拆解法

将文章核心概念拆解为视觉元素：

1. **核心概念** → 用一个形状/结构表达
2. **关系/动作** → 用箭头/流向/对比表达
3. **情绪/氛围** → 用色调/光影/留白表达
4. **细节/层次** → 用纹理/密度/大小变化表达

```
示例："AI 正在重塑教育"
- 核心概念：重塑 = 一个正在变形的几何体（从方块变成球体）
- 关系：AI = 几何体表面的电路纹理
- 情绪：未来感 = 冷色调 + 发光边缘
- 细节：教育 = 几何体周围漂浮的书页碎片
```

### 方法二：视觉对比法

用画面内的对比关系表达概念：

| 对比类型 | 视觉表达 | 示例 |
|---------|---------|------|
| 大小对比 | 巨大 vs 微小 | 一只蚂蚁面对一座山 = 挑战 |
| 明暗对比 | 光明 vs 黑暗 | 一束光穿透黑暗 = 希望/突破 |
| 完整 vs 破碎 | 整体 vs 碎片 | 一面完整的镜子和一地碎片 = 颠覆 |
| 有序 vs 混乱 | 网格 vs 涂鸦 | 左侧整齐右侧混乱 = 失控 |
| 静止 vs 运动 | 凝固 vs 流动 | 冰块与水流 = 变革 |

### 方法三：空间隐喻法

用空间关系表达抽象概念：

| 空间关系 | 概念映射 | 示例 |
|---------|---------|------|
| 中心 vs 边缘 | 核心 vs 外围 | 中心一个发光体，周围是碎片 = 聚焦 |
| 上方 vs 下方 | 高级 vs 基础 | 上方云层，下方地面 = 理想 vs 现实 |
| 内部 vs 外部 | 内在 vs 外在 | 一个打开的盒子，内部是星空 = 潜能 |
| 近 vs 远 | 当下 vs 未来 | 前景清晰，背景模糊 = 短期 vs 长期 |
| 封闭 vs 开放 | 限制 vs 自由 | 一扇打开的门，门外是光 = 突破 |

## 画面构成检查清单

生成后检查：
- [ ] 画面中是否没有任何拟人化形象（无脸、无表情、无动作姿态）？
- [ ] 概念是否可以通过画面本身直接理解（不需要文字说明也能猜出大意）？
- [ ] 画面是否有足够的视觉张力（对比、留白、焦点）？
- [ ] 元素之间是否有清晰的逻辑关系（不是随机堆砌）？
- [ ] 氛围是否与文章情绪一致（严肃/轻松/紧张/希望）？

## 与两种角色模式的对比

| 维度 | 品牌锚定模式 | 情境锚定模式 | 无角色模式 |
|------|------------|------------|----------|
| 画面主体 | 固定角色 | 定制角色 | 概念/物体/场景 |
| 一致性 | 跨文章一致 | 单篇文章内一致 | 无一致性要求 |
| 品牌感 | 强 | 弱 | 无 |
| 内容贴合度 | 中（角色适配主题） | 高（角色为主题定制） | 最高（概念直接表达） |
| 创作自由度 | 低（受角色限制） | 中（受角色约束） | 最高（无任何约束） |
| 适用场景 | 品牌内容、系列文章 | 单篇文章、创意内容 | 数据/方法/概念/氛围类文章 |
| 情感传递 | 通过角色表情动作 | 通过角色设计 | 通过氛围、对比、空间 |

## 使用建议

- **数据/图表类文章** → 优先用方向 A（纯概念隐喻）
- **技术架构/流程说明** → 优先用方向 B（物体叙事）
- **情绪/氛围类文章** → 优先用方向 C（场景氛围）
- **用户说"不要角色""不要人物""纯画面"** → 直接启用无角色模式
- **概念非常抽象难以角色化** → 用无角色模式更合适

## 风格一致性规则（强制）

无角色模式**不是**"每张图可以随意换风格"。同一文章的所有插图必须严格遵守：

1. **风格锁定**：Step 5 选定一种风格后，该文章所有插图必须使用此风格
2. **视觉方向可变**：同一风格下，不同图片可以使用不同的视觉方向（A/B/C）
3. **示例**：选择了"赛博霓虹"风格 → 7 张图全部是赛博霓虹风格，但有的用纯概念隐喻、有的用物体叙事、有的用场景氛围
4. **禁止行为**：同文章中一张用赛博霓虹、一张用木刻版画、一张用基础素描

## 注意事项

- 无角色不等于"无主体"，画面仍需有视觉焦点
- 避免变成"PPT 插图"或" stock photo"风格
- 保持手绘感、草图感，不要过度精致
- 留白依然重要，不要填满画面
- 一图一核心原则不变，不要试图在一个画面里讲多个概念
