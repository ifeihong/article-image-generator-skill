# 视觉基因 (Visual Genome)

## 一句话
纯白、极简、手绘、留白、克制、怪诞、产品草图感、中文手写感、结构清楚但不说明书。
像一个人在白纸上随手画出来的一张解释草图。

## 画幅规范

### 支持的画幅比例

| 比例 | 方向 | 推荐尺寸 | 最佳使用场景 |
|:---:|:---:|:---|:---|
| **16:9** | 横版 | 1280×720 / 1920×1080 | 文章正文配图、公众号、博客、视频封面（默认） |
| **9:16** | 竖版 | 720×1280 / 1080×1920 | 小红书、抖音、手机壁纸、短视频 |
| **4:3** | 横版 | 1024×768 / 1440×1080 | PPT、课件、报告、演示文档 |
| **3:4** | 竖版 | 768×1024 / 1080×1440 | 小红书图文、Instagram、社交媒体 |
| **1:1** | 方形 | 1024×1024 / 1080×1080 | 头像、社交媒体、产品图、方形海报 |
| **21:9** | 超宽 | 2560×1080 / 3440×1440 | 横幅、网站封面、B站头图、宽屏展示 |

### 画幅选择原则

- **同一篇文章的所有配图必须使用相同画幅**，确保视觉统一
- 画幅选择应在消化正文之前完成，一旦选定不可更改
- 用户未指定时，根据内容类型智能推荐（见 SKILL.md Step 2）

### 各画幅的视觉调整

**横版（16:9 / 4:3 / 21:9）**：
- 主体偏左或居中，右侧留白放批注
- 适合展示流程、对比、横向结构

**竖版（9:16 / 3:4）**：
- 主体居中或偏上，下方留白
- 适合展示层级、纵向流程、人物全身

**方形（1:1）**：
- 主体居中，四周均匀留白
- 适合头像、图标、对称构图

## 必须

- **纯白背景**：不要米色、暖灰、纸张纹理、渐变、阴影、噪点、复古纸感
- **黑色手绘线稿为主**：细线、轻微抖动、不机械、不矢量、不厚重描边
- **大量留白**：主体占画面约 40%-60%，至少 35% 空白，最好有一整块安静区域
- **少量手写批注**（语言自适应）：最多 5-8 处，每处简短。中文 2-8 字，英文 1-3 词，日文/韩文 2-6 字
- **一张图只讲一个核心动作、结构、状态或隐喻**
- **结构要自然表达，不要在图上写结构类型名称**

## 颜色

- 黑色：主体线稿、角色、框线、结构、主要文字、主体物件
- 红色：重点批注、问题、情绪点、关键提醒、结果
- 橙色：主流程、路径、箭头、自动化流向、从 A 到 B 的移动关系
- 蓝色：补充说明、脑内状态、系统状态、第二层解释、AI/assistant/自动化提示

蓝色不是每张都必须用。颜色要克制，宁可少不要多。

## 绝对不要

- 不要商业插画
- 不要 PPT 信息图
- 不要正式流程图
- 不要课程课件
- 不要可爱卡通海报
- 不要儿童插画
- 不要复杂架构图
- 不要精致扁平插画
- 不要科技感 UI
- 不要真实 App 截图
- 不要复杂背景、渐变、阴影、纹理
- 不要把每个节点都解释清楚
- 不要左上角写"Workflow 流程图 / 系统架构图 / 常见坑 / 路线图"等类型标题

## 审美方向

要怪诞、创意、有意思、简洁清爽、天马行空。
不要可爱、幼稚、很复杂、死板。
