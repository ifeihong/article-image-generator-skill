## 개요

**Article Image Generator**는 Codex, Trae, Claude Code용으로 설계된 AI Skill입니다. 텍스트 콘텐츠를 시각적으로 인상적인 16:9 가로 일러스트로 변환합니다. 범용 AI 아트 도구나 스톡 사진 검색과 달리, 글의 서사 구조를 이해하고 아이디어의 진정한 시각적 확장이 되는 이미지를 생성합니다 —— 단순한 장식이 아닙니다.

2가지 **캐릭터 앵커 모드**와 10가지 **비주얼 방언**을 중심으로 구축되어 콘텐츠의 톤, 주제, 독자의 기대에 맞춰 창의적 출력을 조정합니다.

> "일러스트는 설명해서는 안 된다 —— 확장해야 한다."

## 주요 기능

### 듀얼 캐릭터 앵커 모드

브랜드 일관성과 콘텐츠 고유 창의성 사이에서 선택:

| 측면 | 브랜드 앵커 모드 | 컨텍스트 앵커 모드 |
|------|----------------|------------------|
| **핵심 로직** | 캐릭터 참조 이미지 업로드, 모든 일러스트에 동일한 형상 사용 | 각 글의 주제에 맞는 독창적 캐릭터 생성 |
| **최적 사용처** | 브랜드 콘텐츠, 시리즈 글, 개인 IP, 기업 블로그 | 심층 글, 창의적 실험, 트렌드 주제 |
| **캐릭터 일관성** | 최대 (모든 콘텐츠 동일 형상) | 높음 (한 글 내 동일 형상) |
| **브랜드 인지도** | 강함 (시각적 시그니처) | 미묘함 (콘텐츠 우선 접근) |
| **창작 자유도** | 기존 캐릭터에 제약 | 무제한, 창의성 최대화 |

### 10가지 비주얼 방언

| ID | 방언 | 시각적 특징 | 최적 사용처 |
|:---:|:---|:---|:---|
| 0 | **베이스 스케치** | 흰 캔버스, 검은 손그림 선, 부조리한 명료함 | 기본 — 테크, 비즈니스, 일반 |
| 1 | **레트로 테크** | 누렇게 변한 공학 매뉴얼, 기계 제도 미학 | 역사, 산업, 향수 |
| 2 | **오가닉 웜스** | 식물 맥, 수채화 번짐, 자연적 성장 | 환경, 웰니스, 라이프스타일 |
| 3 | **미니멀 지오메트리** | 바우하우스 그리드, 정밀 분할, 절제된 색채 | 데이터 시각화, 디자인, 고급 이성 |
| 4 | **믹스드 미디어** | 사진 베이스 + 손그림 오버레이, 실물 매시업 | 튜토리얼, DIY, 창작 과정 |
| 5 | **잉크 워시 선** | 동양 수묵, 여백의 미, 붓 농담 | 문화, 철학, 동양 미학 |
| 6 | **사이버 네온** | 네온 빛 효과, 어두운 배경, 테크 발광 | 미래 기술, AI, 메타버스, 디지털 문화 |
| 7 | **우드컷 프린트** | 대담한 흑백 대비, 칼 조각 텍스처, 민예 | 사회 이슈, 인문, 강력한 표현 |
| 8 | **클레이 스톱모션** | 3D 점토 질감, 손제작, 둥근 매력 | 교육, 아동, 가벼운 과학, 힐링 |
| 9 | **글리치 아트** | 디지털 손상, 스트라이프 변위, 데이터 감소 미학 | 디지털 비판, 해커 문화, 실험 예술 |

## 빠른 시작

### 기본 호출

```
Article Image Generator를 사용하여 이 글의 일러스트를 생성해 주세요.
```

### 모드 지정

```
브랜드 앵커 모드 사용.    ← assets/ip-reference/ 참조 이미지 사용
컨텍스트 앵커 모드 사용.   ← 커스텀 캐릭터 생성
```

### 방언 지정

```
사이버 네온 방언 사용.
```
